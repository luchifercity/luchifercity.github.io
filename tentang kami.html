  • Sejarah Termux

    • Apa itu Termux ?

    Termux merupakan sebuah aplikasi terminal dan juga merupakan linux enviroment berbasis linux debian yg mendukungan perangkat berarsitektur 32 dan juga 64bit dengan minimal Os yg di jalani merupakan android versi 5.0 (lolipop)

    Cara Menggunakan Termux

    Untuk cara menggunakan termux sendiri terbilang sangat mudah sekali yg perlu kalian lakukan pertama kali ialah menginstall aplikasi termux, kalian bisa mendownload termux melalui playstore atau f-store

    Setelah di buka maka secara otomatis termux akan menginstall paket proses penginstallan terbilang sangat cepat bahkan kurang dari 1 menit sangat berbeda sekali dengan Gnu root yg membutuhkan waktu sekitar 5 menitan pada saat proses instalalasi

    Dan tahap selanjutnya setelah proses instalasi ialah melakukan update dan upgrade agar mendapatkan package terbaru yg di sediakan oleh termux kalian bisa melakukan perintah berikut apt update && apt upgrade pada terminal

    Fungsi Termux

    Jika di tanya Fungsi dan Keggunaan Termux, seperti yg sudah saya sebutkan di atas di samping sebagai terminal Termuxsecara garis besar merupakan sebuah distro linux yg berbasis debian

    Dan berikut saya rangkum beberapa fungsi Termux

    1. Termux bisa di gunakan sebagai SSH Client

    2. Termux bisa di gunakan untuk menjalankan Python, Ruby, Perl, tanpa perlu akses root

    3. Termux bisa di gunakan untuk mengakses internet dengan lynx yg mana merupakan browser berbasis terminal

    4. Termux bisa di gunakan sebagai aplikasi teks editor berbasis terminal

    5. Termux bisa di gunakan sebagai media untuk membobol jaringa wifi

    Dan masih banyak lagi fungsi dan keggunaan lainnya yg bisa di lakukan oleh termux sendiri yg mana jika saya jelaskan satu persatu akan sangat panjang dan memakan waktu

    Perintah Termux

    "Sebagian besar perintah yg di jalankan pada distro debian bisa di terapkan pada termux, mungkin hanya perintah user root dan instalasi aplet yg membedakannya jika biasanya kita menginstall aplet dengan perintah apt-get install (nama aplet) pada distro debian maka pada termux kita cukup menggunakan perintah apt install (nama aplet) atau bisa juga dengan perintah pkg install (nama aplet)".

    "Kemudian apabila ingin melakukan login sebagai user root perintah yg di jalankan pada umumnya ialah sudo maka beda halnya dengan Termux jika ingin login dengan user root kita bisa menggunakan perintah tsu jangan lupa menginstall aplet tersebut menggunakan perintah apt install tsu dengan syarat perangkat harus sudah dalam keadaan root".
